1. ОБЩЕЕ ОПИСАНИЕ
Жанр: Фэнтези-платформер с процедурной генерацией

Платформа: PC, консолы

Количество игроков: 1

Режим: Одиночное прохождение

2. ТРЕБОВАНИЯ К ГЕЙМПЛЕЮ
2.1. Базовые механики
Управление одним персонажем со способностью управления гравитацией.

Физика движения и падения, привязанная к вектору гравитации.

Система "Хранилища Бездны" (инвентарь для ключевых предметов и ресурсов).

Механика "перемагничивания" гравитации для определенных платформ.

2.2. Система гравитации
Требования к гравитационному движку:

Реализация мгновенного перенаправления вектора гравитации для игрока.

Расчет новой "опорной поверхности" при смене гравитации.

Корректное отображение мира и UI при любом векторе гравитации.

Визуализация зон, где смена гравитации невозможна.

Уровни управления гравитацией:

Локальная гравитация: Стандартная способность. Игрок "падает" на ближайшую поверхность в выбранном направлении (Вверх/Вниз/Влево/Вправо).

Инерционная гравитация: После смены вектора игрок сохраняет часть инерции от предыдущего движения, позволяя делать дальние "броски".

Магнитная гравитация: Позволяет на короткое время "прилипнуть" к определенным металлическим платформам, игнорируя общий вектор гравитации.

2.3. Искусственный интеллект
Поведение врагов (Стражи Руин):

Патрулирование по заданным маршрутам, привязанным к своей гравитации.

Реакция на визуальное обнаружение игрока в их плоскости гравитации.

Поиск игрока в последней известной точке появления.

Неспособность менять гравитацию, но использование стационарных переходов.

Состояния AI:

Патрулирование: Движение по заданному циклу.

Тревога: Ускорение движения и расширение угла обзора при подозрении на игрока.

Преследование: Прямая погоня за игроком в пределах своей гравитационной плоскости.

Поиск: Исследование последней известной позиции игрока в течение ограниченного времени.

3. СИСТЕМА ПЕРСОНАЖА
3.1. Хранитель (Главный герой)
Способности:

Сердце Бездны (Базовая способность): Мгновенная смена личного вектора гравитации.

Взор Бездны: Кратковременное замедление времени для оценки обстановки и точного выбора вектора гравитации (ограниченный заряд).

Прыжок Инверсии: Короткий телепорт-рывок в направлении, противоположном текущей гравитации, для быстрого уклонения.

Чувство Гравитации: Пассивная способность, подсвечивающая контуры платформ в направлении текущего вектора гравитации.

Ограничения:

Не может атаковать напрямую, только уворачиваться и использовать окружение.

Способность смены гравитации имеет короткий период восстановления (кулдаун).

Некоторые зоны "Застойной магии" блокируют использование способностей.

4. СИСТЕМА РЕСУРСОВ И УЛУЧШЕНИЙ
4.1. Типы ресурсов
Эссенция Бездны: Восстанавливает заряд "Взора Бездны" и "Прыжка Инверсии".

Осколки Времени: Постоянное улучшение характеристик (уменьшение кулдауна, увеличение длительности замедления).

Древние Руны: Открывают новые способности (например, "Магнитная гравитация").

Ключи Пространства: Одноразовые предметы для активации порталов и специальных механизмов.

4.2. Улучшения способностей
Доступные улучшения:

Ускоренное Восстановление: Уменьшает кулдаун смены гравитации.

Увеличенный Запас: Повышает максимальный заряд "Взора Бездны".

Цепная Инверсия: Позволяет сделать две смены гравитации в воздухе без касания поверхности.

Стабильное Притяжение: Дает иммунитет к эффектам гравитационных вихрей.

5. ДИЗАЙН УРОВНЕЙ
5.1. Структура уровней
Процедурная сборка из кубических "Секторов": Каждый сектор — это куб, на всех шести гранях которого расположены платформы, препятствия и враги.

Нелинейное прохождение: Игрок может выбирать, через какую грань сектора перемещаться дальше.

Обязательные гравитационные головоломки: Для прогресса требуется использовать смену гравитации в определенной последовательности.

Скрытые зоны: Секретные комнаты, доступные только при использовании продвинутых техник смены гравитации.

5.2. Гравитационные зоны
Зоны Стабильной Гравитации: Стандартные сектора.

Зоны Хаотичной Гравитации: Сектора, где вектор гравитации случайным образом меняется для игрока с определенной периодичностью.

Зоны Застойной Магии: Блокируют использование способностей, оставляя только базовый прыжок.

Гравитационные Вихри: Перемещают игрока в случайном направлении при контакте.

6. ИНТЕРФЕЙС ПОЛЬЗОВАТЕЛЯ
6.1. Элементы HUD
Индикатор Текущего Вектора: Стрелка в центре экрана, всегда показывающая направление "вниз" для персонажа.

Индикатор Заряда Способностей: Уровень заряда для "Взора Бездны" и "Прыжка Инверсии".

Индикатор Кулдауна: Визуальное отображение времени до следующей возможности смены гравитации.

Мини-карта Пространства: 3D-условная карта, показывающая контуры текущего сектора и смежных с ним.

6.2. Визуальные эффекты
Эффект Смены Гравитации: Плавное затемнение/осветление экрана, изменение ориентации HUD.

Визуализация "Чувства Гравитации": Подсветка активных платформ в поле зрения.

Шейдеры Искажения: Эффекты на границах экрана при нахождении в зонах хаотичной гравитации.

7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
7.1. Графика
Поддержка динамического переориентирования камеры и освещения.

Система частиц для эффектов смены гравитации и телепортации.

Шейдеры для визуализации гравитационных полей и "запретных зон".

Пост-обработка для создания сюрреалистической, "невесомой" атмосферы.

7.2. Производительность
Стабильные 120 FPS при быстрой смене камеры и сложной геометрии уровней.

Оптимизация системы расчета коллизий при любом векторе гравитации.

Эффективное использование памяти для процедурной генерации и подгрузки секторов.

8. СЕТИ И СОХРАНЕНИЯ
8.1. Система сохранений
Контрольные точки (Анкеры Стабильности): Активируются игроком, закрепляя прогресс.

Сохранение коллекционных улучшений: Найденные руны и осколки сохраняются навсегда.

Случайное семя уровня: Возможность сохранить и поделиться seed'ом для генерации конкретного понравившегося уровня.

Статистика прохождения: Запись времени, количества смертей и использований смены гравитации.
